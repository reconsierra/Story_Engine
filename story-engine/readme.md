Story Engine – Personal GitHub SetupThis repository contains a deterministic, preset‑driven story engine designed for personal use. It is structured so you can:Generate books using engine defaults (no configuration)Apply preset packs (Relentless, Slow Burn, etc.)Override one variable at a time when neededAdd new presets later without breaking anythingNothing here is random by default. All decisions are intentional and rule‑driven.1.
